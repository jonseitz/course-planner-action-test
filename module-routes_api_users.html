(route) Add New User

Route:
MethodPath
POST /api/users/new
Adds a new user defined in the request body
Authentication

Requires CAS Authentication. User must have "Admin" permissions.

Body Parameters:
Name Type Attributes Default Description
HUID String The user's 8-digit HUID
firstName String The users' first name
lastName String The user's last name
email String The user's harvard email address
accessLevel String optional "Read-Only" The user's permissions in the app. Must be one of "Read-Only", "Privileged", or "Admin"
